<h2 id="overview">Overview</h2>

<p align="center">
Finance Buddy is an all-in-one developer toolkit for building scalable, accessible, and visually consistent financial applications. It combines a comprehensive set of React components, styling utilities, and API integrations to streamline development and enhance user experience.
</p>

---

<h2 id="why-finance-buddy" align="center">Why Finance Buddy?</h2>

<p align="center">
This project aims to simplify the creation of modern financial platforms by providing a cohesive design system and powerful features. The core features include:
</p>

<ul align="center">
  <li>🎨 <strong>Customizable Theming:</strong> Seamless theme switching with Tailwind CSS and dark mode support.</li>
  <li>⚙️ <strong>Reusable UI Components:</strong> Buttons, modals, forms, charts, and navigation elements built for accessibility and responsiveness.</li>
  <li>🔧 <strong>Utility Functions:</strong> Class merging, device detection, and toast notifications for efficient UI management.</li>
  <li>🤖 <strong>AI-Driven APIs:</strong> Integrations with Google Generative AI for personalized investment advice and financial guidance.</li>
  <li>📊 <strong>Data Visualization:</strong> Interactive charts and dashboards for financial insights.</li>
  <li>🧩 <strong>Modular Architecture:</strong> Components designed for scalability, maintainability, and seamless integration.</li>
</ul>
